# RECIPE_INDEX.md

This is a centralized tag-based reference linking to all modular recipe files across the Luna System.

---

## 🏷️ Tagging Systems

### 🍽 BASSSSS Tags — Flavor Profile
Use these to help balance meals across a week or plan for mood/season:
- `#bitter`
- `#acid`
- `#sweet`
- `#salty`
- `#spicy`
- `#savory`
- `#sour`

### 👶 T-Rating — Toddler Likeability Scale
Use `#T-1` to `#T-5` to reflect how toddler-friendly a recipe is:
- `#T-1` — Major meltdown
- `#T-2` — Nibbled and rejected
- `#T-3` — Neutral or polite bites
- `#T-4` — Enjoyed, maybe asked for more
- `#T-5` — Favorite / Clean plate

### ⏱ Utility Tags
- `#previousDayPrep` — Requires prep 8+ hours before (e.g., defrost, soak)
- `#under20` — Can be prepped and cooked in under 20 minutes
- `#mealprep` — Great for batch cooking or storing
- `#pantryfriendly` — Uses mostly shelf-stable or common items
- `#leftoverfriendly` — Tastes good next day or cold

- Recipe: [Recipe Name]

  Tags: #BASSSS #T-rating #customTags
  Notes: Optional description or dietary considerations.
